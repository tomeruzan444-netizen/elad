# אתר אלעד שורתי — ייעוץ ופיתוח עסקי

אתר סטטי מהיר הבנוי ב-[Astro](https://astro.build), מיגרציה מ-WordPress עם שמירה מלאה על תוכן, מבנה URL ו-SEO.

## פקודות

| פקודה | מה היא עושה |
|-------|-------------|
| `npm install` | התקנת תלויות (פעם אחת) |
| `npm run dev` | שרת פיתוח מקומי ב-`localhost:4321` |
| `npm run build` | **בניית פרודקשן** (לאתר החי — מאונדקס בגוגל) |
| `npm run build:staging` | **בניית גרסה זמנית** — חסומה לאינדוקס (`noindex` + `robots Disallow`) |
| `npm run preview` | תצוגה מקומית של תיקיית `dist` |

תוצרי הבנייה נמצאים בתיקיית `dist/` — תיקייה של קבצי HTML/CSS/JS סטטיים שאפשר להעלות לכל שרת.

## חשוב: אינדוקס

- לדומיין **זמני** → תמיד `npm run build:staging` (האתר לא ייכנס לגוגל).
- לדומיין **הסופי** (elad-digital.co.il) → `npm run build` (האתר יאונדקס).

ראו `DEPLOY.md` למדריך פריסה מלא.
